# Dark/Light Mode Quick Reference Card

## Device Settings at a Glance

| Device | Display Mode | Icons | Night Shift | Auto Brightness | Special Settings |
|--------|-------------|-------|-------------|-----------------|------------------|
| **MacBook Air M1** | Light only | Full-color | On (default) | On or 40-50% | True Tone on |
| **iPhone 13** | Auto dark | Full-color | On (default) | On | Reduce White Point toggle |
| **iPad mini 6** | Auto dark | Clear (Auto) | On (default) | On | Triple-click: White Point |
| **External Display** | Light only | N/A | On (default) | N/A | None |

## Environment Rules

**Office (Docked MacBook):**
- Light mode always
- Warm overhead bulbs (2700K)
- No laptop mobility = no mode switching

**Mobile MacBook (Creative Work):**
- Light mode always
- Ensure well-lit space or reduce brightness to 30%
- Night Shift handles evening warmth

**iPhone (Minimalist Use):**
- Auto dark mode handles time-of-day
- Reader Mode for reading
- Dark Night extension for backup

**iPad (Consumption):**
- Auto dark mode + clear icons = visual calm
- Use lamp when reading in dim light
- Triple-click white point toggle as needed

## Key Principles

✅ **Light mode = clarity** for knowledge work
✅ **Auto dark mode = comfort** for consumption/evening
✅ **Night Shift default schedule** handles circadian rhythm
✅ **No browser extensions** needed (native settings sufficient)
✅ **Predictable rules** reduce ADHD decision friction
✅ **Context-dependent** not uniform

## Troubleshooting

**Too bright at night?** 
- Enable Reduce White Point (triple-click)
- Lower brightness to 30%
- Check ambient lighting

**Icons too abstract?**
- iPad: Switch from Clear to Tinted or Full-color
- iPhone: Keep full-color (speed matters)

**Eyes dry?**
- Reduce screen brightness to 40-50%
- Use 20-20-20 rule
- Add desk lamp (warm 2700K)

**Battery draining?**
- iPhone dark mode saves 3-9% (marginal)
- iPad LCD = no savings
- Don't sacrifice clarity for minimal battery gain
