# Setup Checklist: Dark/Light Mode Optimization

## MacBook Air M1 Setup

### Display Settings
- [ ] System Settings > Appearance > **Light**
- [ ] System Settings > Displays > Night Shift > Schedule: **Sunset to Sunrise**
- [ ] System Settings > Displays > True Tone > **On** (optional)
- [ ] System Settings > Displays > Automatically adjust brightness > **On** (or manually set 40-50%)

### Environment
- [ ] Replace office overhead bulbs with **warm 2700K** bulbs (not daylight)
- [ ] Ensure well-lit workspace (no working in dim lighting with light mode)
- [ ] Optional: Add warm desk lamp for evening creative work

### Browser Settings
- [ ] No extensions needed (native Night Shift handles everything)
- [ ] Verify browser respects system light mode

---

## iPhone 13 Setup

### Display Settings
- [ ] Settings > Display & Brightness > **Automatic** (toggle on)
- [ ] Settings > Display & Brightness > **Dark** appearance (system will auto-schedule)
- [ ] Settings > Display & Brightness > Night Shift > Schedule: **Sunset to Sunrise**
- [ ] Settings > Display & Brightness > Automatically adjust brightness > **On**

### Accessibility Shortcuts
- [ ] Settings > Accessibility > Accessibility Shortcut > Select **Reduce White Point**
- [ ] Test: Triple-click side button should toggle Reduce White Point

### Safari Extensions
- [ ] Install **Dark Night** extension (App Store) as backup for bright websites
- [ ] Primary tool: Use **Reader Mode** (tap AA icon in Safari address bar)

### Icon Style
- [ ] Keep **full-color icons** (default)
- [ ] Do NOT switch to Clear or Tinted (speed matters for minimalist use)

---

## iPad mini 6 Setup

### Display Settings
- [ ] Settings > Display & Brightness > **Automatic** (toggle on)
- [ ] Settings > Display & Brightness > **Dark** appearance (system will auto-schedule)
- [ ] Settings > Display & Brightness > Night Shift > Schedule: **Sunset to Sunrise**
- [ ] Settings > Display & Brightness > Automatically adjust brightness > **On**

### Icon Customization
- [ ] Long-press home screen until icons jiggle
- [ ] Tap **Edit** > **Customize**
- [ ] Select **Clear** icons
- [ ] Choose **Auto** at bottom (switches Clear Light/Dark with system appearance)

### Accessibility Shortcuts
- [ ] Settings > Accessibility > Accessibility Shortcut > Select **Reduce White Point**
- [ ] Test: Triple-click top button should toggle Reduce White Point

### Environment
- [ ] Keep lamp nearby for reading in dim lighting
- [ ] Test brightness: Reduce to 30-40% if light mode feels too bright at night

---

## Troubleshooting Checklist

### "Light mode too bright on MacBook"
- [ ] Check brightness: Should be 40-50% in office
- [ ] Verify warm overhead lighting (not daylight bulbs)
- [ ] Disable True Tone if causing issues

### "iPhone still feels bright at night"
- [ ] Verify auto dark mode enabled and scheduled
- [ ] Enable Reduce White Point (triple-click)
- [ ] Check Night Shift is on default schedule

### "iPad light mode too harsh"
- [ ] Switch to auto dark mode (done âœ“)
- [ ] Enable Reduce White Point toggle (done âœ“)
- [ ] Use lamp when reading in dim environments

### "Can't find apps with Clear icons on iPad"
- [ ] Switch to Tinted icons (Settings > Home Screen > Customize > Tinted)
- [ ] Or switch back to full-color icons
- [ ] Clear icons are aesthetic choice, not functional requirement

### "Browser websites too bright on MacBook at night"
- [ ] Night Shift should handle this automatically
- [ ] If still jarring: Reduce brightness (not mode)
- [ ] Consider this a signal to wind down screen time
